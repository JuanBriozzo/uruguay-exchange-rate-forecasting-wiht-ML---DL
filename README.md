# uruguay-exchange-rate-forecasting-wiht-ML---DL
This study examines UYU/USD exchange rate forecasting in Uruguay using ML and DL techniques. Using a dataset with more than 140 local, regional, and global variables for 2015–2025, it trains and evaluates tree-based models, support vector regression, and recurrent and hybrid neural networks over 1, 3, 6, and 12-month horizons.
